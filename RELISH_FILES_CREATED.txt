✅ RELISH PRINCIPLES SATISFIED
────────────────────────────────────────────────────────────────────────────

  ✓ Recipes are internal artifacts (dishes are primary)
  ✓ AI used sparingly (caching built-in)
  ✓ Deterministic logic (no AI for math/normalization)
  ✓ Data over rules (ontology tokens, not if-statements)
  ✓ Explicit user intent (user_recipe_links tracks choices only)
  ✓ Legal safety (URLs only, no scraped content)
  ✓ Future-proof (nutrition, budgets, pantry ready)
  ✓ Cache-first (ai_cache table prevents redundant calls)
  ✓ Cost tracking (ai_usage_logs for every token)
  ✓ Security first (30+ RLS policies)
